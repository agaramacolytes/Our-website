# Image Setup Instructions

Your website is now configured to use professional images. Here's how to add them:

## Image Files Needed

1. **professional-image.jpg** - Your professional photo (place in `/public` folder)
   - Recommended size: 500x500px or larger
   - Used in the About section

2. **project-1.jpg, project-2.jpg, project-3.jpg** - Portfolio project images (place in `/public` folder)
   - Recommended size: 600x400px
   - Used in the Portfolio/Recent Projects section

**Note:** The logo/brand name now uses "Agaram Acolytes" with the professional Playfair Display serif font. No logo image file is needed.

## Where to Get Professional Images

### Free Image Resources:
- **Unsplash** (https://unsplash.com) - Free high-quality photos
- **Pexels** (https://pexels.com) - Free stock photos
- **Pixabay** (https://pixabay.com) - Free stock images
- **Google Images** - Search for professional images and download
- **Canva** (https://canva.com) - Create professional images

### Search Keywords:
- For professional photo: "professional headshot", "business professional portrait", "software developer portrait"
- For project images: "web design portfolio", "e-commerce website", "modern dashboard", "SaaS application"

## How to Add Images

1. Download your chosen images
2. Resize them if needed to the recommended sizes
3. Save them in the `/public` folder with the exact filenames mentioned above
4. The changes will automatically reflect on your website at http://localhost:3000
